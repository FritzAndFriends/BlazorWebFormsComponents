## Key Learnings

1. **Auth endpoints must be consolidated** тАФ Having both ProgramCsEmitter and RedirectHandlerAnnotator generate auth endpoints causes duplicates
2. **EF Core cannot translate `string.Equals(a, b, StringComparison.X)`** тАФ Use simple `==` for SQL Server (case-insensitive by default)
3. **SSR pages cannot use NavigationManager.NavigateTo for redirects** тАФ Use `HttpContext.Response.Redirect()` instead
4. **InteractiveServer pages with `prerender: false`** render blank until SignalR connects тАФ tests must wait for content
5. **Registration auto-signs-in** тАФ No need to test login separately if registration includes SignInAsync
